Getting started

Unzip the archive and find out the folder www. There are all the template files in this folder. You can open the following files for viewing in browser.

  1. /css — folder with css files.
  2. /fonts — folder with iconfont.

  4. /img — folder with image files.
  5. /js — folder with javascript files.

  6. index.html — home page.

HTML structure

In my template I use Bootstrap grid, if you don't know how to use it, here is a detailed information on its use.

The standard section has the following HTML structure:

<section class="row row--grid">
  <div class="col-12">
    <div class="main__title">
      <h2>Title</h2>
      <a href="#" class="main__link">Link</a>
    </div>
  </div>

  <div class="col-12">
    ...
  </div>
</section>

Logotype

To change logotype, find image img/logo.svg and replace it with your logo.

Audio player

I recommend that you familiarize yourself with the documentation for Plyr. There you can find detailed information about how the plugin works.


Single audio track – any link with data-link attribute and also with attributes data-title, data-artist, data-img for display in the audio player.

<a data-link data-title="Track" data-artist="Artist" data-img="img/img.jpg" href="audio/track.mp3">
    ...
</a>

Playlist – links with data-playlist attribute inside container with class main__list--playlist

<ul class="main__list--playlist">
  <li>
    <a data-playlist data-title="Track" data-artist="Artist" data-img="img/img.jpg" href="audio/track.mp3">
        ...
    </a>
  </li>
  <li>
    <a data-playlist data-title="Track" data-artist="Artist" data-img="img/img.jpg" href="audio/track.mp3">
        ...
    </a>
  </li>
      ...
</ul>
